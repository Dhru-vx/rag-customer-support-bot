# 🛒 E-Commerce RAG Support Bot

An intelligent customer support chatbot built using **Retrieval-Augmented Generation (RAG)** with LangChain, FAISS, and Streamlit.

This chatbot retrieves relevant policy information using vector similarity search and provides structured responses without relying on external LLM APIs.

---

## 🚀 Project Overview

This project demonstrates how to build a production-style RAG system using:

- 📄 Policy-based document retrieval
- 🔍 FAISS vector similarity search
- 🧠 Local embeddings (Sentence Transformers)
- 💬 Interactive Streamlit chat interface
- ⚡ Manual RAG pipeline (LangChain v1 compatible)
- ❌ No external LLM API required
